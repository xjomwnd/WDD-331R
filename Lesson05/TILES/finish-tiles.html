Creating a tiled media gallary

Pinterest really popularized the tiled media layout. One of the challenges with Pinterest is that they don't have control over what size the card will be. Or rather they gave up some control. They do constrain the width, but the height can vary. This activity will have you use Flexbox to create a tiled layout in both a row and a column.

Objective:

Using CSS Flexbox, create a tiled layout similar to the provided example.

Instructions

  1. Grab the starter code.

    Download this starter code and open it in your editor. It contains a bit of Javascript to randomly generate a bunch of different sized media elements. Note that there are two sets...the first class="row-grid" will all have the same height. The second class="col-grid" should all have the same width.

    Everytime you refresh the page you will get a different set of cards.

  2. Style the row grid

    Use flexbox to style the row-grid to match the example below.
