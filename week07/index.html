Guidelines to speed up your website

Once you have tested the speed of your website, you can start optimizing it.

1. Use a Content Delivery Network (CDN)

With CDN, user requests are redirected to the nearest server. As a result, the content is delivered to a user quicker and a website works faster. This is a rather expensive, but quite effective way to optimize the load time.

2. Hosting

Virtual Private Servers (VPS) hosting is the best

3. Reduce the image size

Another way to reduce the image size is to use the HTML responsive images and attributes that adjust image size based on user display properties.

4. Reduce the number of plugins

Try to avoid plugins that load a lot of scripts and styles or generate a lot of database queries.

5. Minimize the number of JavaScript and CSS files

There are a lot of tools to minify HTML, CSS, and JavaScript files quickly. For instance, you can use WillPeavy, Script Minifier, or Grunt tools.

6. Use website caching

You can use the following plugins: W3 Total Cache or W3 Super Cache.

7. Use of Gzip

Gzip Compression is an effective way to reduce the size of files.

8. Reduction of Data Base

Database optimization is the an effective way to increase performance.

9. Reduce number of fonts

Choose only the needed styles

11. Reduce redirects

Website redirects create additional HTTP requests which negatively impact performance.

12. Use prefetching techniques

Prefetching entails reading and executing instructions before a user initiates them. The technique is rather common.
